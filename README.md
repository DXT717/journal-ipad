# Inner Room Journal

A private, iPad-friendly journaling web app with mood tracking, prompts, Apple Pencil text input, sketch notes, local saving, export, and supportive reflective replies.

## Publish

This folder is a static website. Upload the contents of this folder to any static host, such as Netlify, Vercel, GitHub Pages, or Cloudflare Pages.

For the easiest path:

1. Open https://app.netlify.com/drop
2. Drag this `journal-ipad` folder onto the page.
3. Open the generated website URL on your iPad.
4. In Safari, tap Share, then Add to Home Screen.

Entries are saved privately in the browser storage on the device where you write them.
